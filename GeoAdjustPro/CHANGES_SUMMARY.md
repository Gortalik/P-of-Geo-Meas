# 📋 Сводка изменений P-of-Geo-Meas

## ✅ Выполненные изменения

### 1. Приветственное окно (WelcomeDialog)
**Файл:** `src/geoadjust/gui/welcome_dialog.py` (создан новый)

**Функциональность:**
- Появляется сразу после запуска приложения
- Две основные кнопки: "Создать новый проект" и "Открыть проект"
- Отображение списка недавних проектов (до 5)
- Информация о версии и описание системы
- Современный дизайн с градиентным фоном
- Три сигнала для обработки действий:
  - `new_project_requested`
  - `open_project_requested`
  - `recent_project_requested`

### 2. Точка входа приложения (__main__.py)
**Файл:** `src/geoadjust/__main__.py` (обновлён)

**Изменения:**
- Интеграция приветственного диалога при запуске
- Обработчики сигналов для создания/открытия проектов
- Создание менеджера проектов для управления жизненным циклом
- Автоматическое создание главного окна после выбора действия
- Поддержка открытия недавних проектов

### 3. Проект с настройками по умолчанию
**Файл:** `src/geoadjust/io/project/project_manager.py` (обновлён)

**Новый метод:** `_set_default_settings(project)`

**Настройки по умолчанию:**
1. **СК-42, зона 7** (`crs.json`)
   - base_crs: SK42
   - zone: 7
   - central_meridian: 39.0
   - false_easting: 7500000.0

2. **Библиотека приборов** (`instruments.json`)
   - Leica TS16 (угловая точность 1", дистанционная 1+1ppm)
   - Trimble M3 (угловая точность 3", дистанционная 2+2ppm)

3. **Параметры уравнивания** (`adjustment.json`)
   - method: classic (классический МНК)
   - max_iterations: 10
   - convergence_threshold: 1e-6

4. **Параметры предобработки** (`preprocessing.json`)
   - check_circle_closure: True
   - check_reciprocal_measurements: True
   - apply_atmospheric_corrections: True
   - apply_refraction: True

5. **Нормативные классы** (`normative_classes.json`)
   - 4 класс (невязка 5", относительная 1/25000)
   - 1 разряд (невязка 10", относительная 1/10000)

## 🎯 Последовательность работы приложения

1. **Запуск** → Отображение приветственного окна
2. **Выбор действия:**
   - "Создать новый проект" → Проект с настройками по умолчанию
   - "Открыть проект" → Диалог выбора файла .gad
   - Выбор из недавних → Открытие последнего проекта
3. **Главное окно** → Полнофункциональный интерфейс с проектом
