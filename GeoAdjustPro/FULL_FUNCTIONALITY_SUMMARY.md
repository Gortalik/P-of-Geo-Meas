# GeoAdjust Pro - Полная функциональность

## Обзор

GeoAdjust Pro теперь является полностью самодостаточной программой для обработки геодезических измерений с полным набором функций, аналогичным профессиональным решениям (КРЕДО ДАТ, Trimble Business Center и др.).

## Интерфейс

### Главное окно

#### Меню
- **Файл**: Создать, Открыть, Сохранить, Свойства, Параметры, Выход
- **Редактирование**: Отменить, Повторить, Копировать, Вставить, Добавить пункт, Добавить измерение
- **Вид**: Схема сети, Панели (Пункты, Измерения, План, Журнал, Свойства)
- **Обработка**: Предобработка, Классический МНК, Робастное уравнивание
- **Отчёты**: Ведомость координат, Ведомость поправок, Отчёт по ГОСТ
- **Справка**: Содержание, О программе

#### Панели
- **Пункты ПВО**: Таблица всех пунктов с возможностью редактирования
- **Измерения**: Таблица всех измерений с возможностью редактирования
- **Ходы и секции**: Дерево ходов и секций
- **План**: Графическое представление сети
- **Журнал**: Лог всех операций
- **Свойства**: Свойства выбранного объекта

#### Строка состояния
- Информация о текущем проекте
- Текущий режим работы
- Прогресс-бар для длительных операций
- Координаты курсора на плане
