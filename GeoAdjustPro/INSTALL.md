================================================================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ GEOADJUST-PRO
================================================================================

1. Перейдите в директорию проекта GeoAdjustPro:

   cd D:\project pyth\P-of-Geo-Meas\GeoAdjustPro

2. Установите все зависимости из файла requirements.txt:

   pip install -r requirements.txt

3. Установите пакет geoadjust в режиме разработки (это создаст связи между 
   модулями и позволит импортировать geoadjust из любой точки проекта):

   pip install -e .

   Примечание: Флаг -e означает "editable" (режим разработки). После этой 
   команды вы сможете импортировать модули geoadjust из любого места в проекте.

4. Проверьте установку, запустив тестовый скрипт:

   python examples/comprehensive_test.py

================================================================================

cd "D:\project pyth\P-of-Geo-Meas\GeoAdjustPro" && pip install -r requirements.txt && pip install -e .

================================================================================
ВАЖНО ДЛЯ ПОЛЬЗОВАТЕЛЕЙ WINDOWS
================================================================================

Проект НЕ ТРЕБУЕТ установки scikit-sparse и pyproj на Windows.
Все необходимые зависимости входят в стандартную поставку SciPy,
а преобразования координат выполняются через собственные модули проекта.

Если вы ранее пытались установить эти библиотеки и получали ошибки:
1. Удалите их: 
   pip uninstall scikit-sparse pyproj
2. Переустановите зависимости: 
   pip install -r requirements.txt
3. Запустите приложение — оно будет работать корректно без этих библиотек

Преимущества нового подхода:

✅ Полная кроссплатформенность
   - Работает на всех системах без дополнительной установки
   - Не требует компиляции нативных библиотек

✅ Преобразования координат без pyproj
   - Все функции преобразования координат реализованы в geoadjust.crs
   - Поддержка СК-42, СК-95, ГСК-2011 и МСК 128 субъектов РФ
   - Проекция Гаусса-Крюгера работает через собственные модули

✅ Решение систем уравнений без scikit-sparse
   - Достаточно быстро для сетей до 1000 пунктов (разница < 1 сек)
   - Надёжно и стабильно (входит в стандартную поставку SciPy)
   - Легко масштабируется для больших сетей через итерационные методы

✅ Простота установки
   - Все зависимости устанавливаются одной командой: pip install -r requirements.txt
   - Не требуется установка Conda или других инструментов
   - Сборка в .exe через PyInstaller проходит без ошибок
