ИНСТРУКЦИЯ ПО УСТАНОВКЕ P-OF-GEO-MEAS
=====================================

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
---------------------
• Windows 10/11 (64-bit) или Linux (Ubuntu 20.04+, Fedora 35+)
• Процессор: Intel Core i3 или эквивалент
• Оперативная память: 4 ГБ (рекомендуется 8 ГБ)
• Свободное место на диске: 500 МБ
• .NET Framework 4.8 (для Windows, устанавливается автоматически при необходимости)
• Python 3.8+ (только для версии без установщика)

УСТАНОВКА В WINDOWS:
--------------------

ВАРИАНТ 1: Через установщик (рекомендуется)
1. Запустите файл P-of-Geo-Meas-Setup-1.0.0.exe
2. Следуйте инструкциям мастера установки
3. После завершения установки запустите приложение из меню "Пуск"
   или с ярлыка на рабочем столе

ВАРИАНТ 2: Портативная версия
1. Распакуйте архив P-of-Geo-Meas-1.0.0-portable.zip в любую папку
   (например, C:\Programs\P-of-Geo-Meas\)
2. Запустите файл P-of-Geo-Meas.exe
3. Приложение готово к работе без установки

УСТАНОВКА В LINUX:
------------------

ВАРИАНТ 1: Через pip
1. Установите зависимости:
   sudo apt-get install python3-pip python3-pyqt5 python3-numpy python3-scipy

2. Установите приложение:
   pip3 install P-of-Geo-Meas

3. Запустите приложение:
   pofgeomeas

ВАРИАНТ 2: Из исходников
1. Клонируйте репозиторий:
   git clone https://github.com/geoadjust/P-of-Geo-Meas.git
   cd P-of-Geo-Meas

2. Установите Poetry (если не установлен):
   curl -sSL https://install.python-poetry.org | python3 -

3. Установите зависимости:
   poetry install

4. Запустите приложение:
   poetry run pofgeomeas

ПЕРВЫЙ ЗАПУСК:
--------------
• При первом запуске приложение создаст папку настроек:
  - Windows: C:\Users\<Ваше имя>\AppData\Roaming\P-of-Geo-Meas
  - Linux: ~/.config/P-of-Geo-Meas

• Рекомендуется настроить параметры проекта в меню "Свойства проекта"

• Проверьте доступные системы координат в меню "Настройки" → "СК"

ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ:
-----------------------------

Проблема: Приложение не запускается
Решение: 
  - Убедитесь, что установлены все системные требования
  - Проверьте наличие .NET Framework 4.8 (для Windows)
  - Запустите от имени администратора

Проблема: Ошибка загрузки ресурсов
Решение:
  - Убедитесь, что все файлы распакованы в одну папку
  - Не перемещайте отдельные файлы после установки

Проблема: Ошибка импорта данных
Решение:
  - Проверьте формат импортируемых файлов
  - Убедитесь, что файлы не повреждены

Проблема: Медленная работа
Решение:
  - Закройте другие приложения
  - Увеличьте объем оперативной памяти
  - Проверьте свободное место на диске
