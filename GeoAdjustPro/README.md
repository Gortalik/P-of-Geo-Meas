# GeoAdjust Pro

**Профессиональная система уравнивания геодезических сетей**

GeoAdjust Pro — профессиональная система уравнивания геодезических сетей, предназначенная для обработки реальных полевых геодезических измерений с максимальной точностью и соответствием нормативным требованиям Российской Федерации.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
# Основные зависимости
pip install -r requirements.txt

# Или для разработки (включает основные + инструменты разработки)
pip install -r requirements-dev.txt
```

### Установка приложения

#### Режим разработки (рекомендуется)

```bash
# Клонирование репозитория
git clone https://github.com/geoadjust/geoadjust-pro.git
cd geoadjust-pro

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Установка в режиме разработки
pip install -e .
```

#### Обычная установка

```bash
pip install geoadjust
```

## 🎯 Запуск приложения

### После установки в режиме разработки

```bash
# Активация виртуального окружения (если не активировано)
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Запуск приложения (вариант 1)
python -m geoadjust

# Запуск приложения (вариант 2)
geoadjust
```

### После обычной установки

```bash
geoadjust
```

### Запуск без установки (из корня проекта)

```bash
python src/geoadjust/__main__.py
```

## 📋 Требования

- **Python**: 3.8 или выше
- **Операционная система**: Windows, Linux, macOS
- **Память**: минимум 2 ГБ ОЗУ
- **Место на диске**: минимум 500 МБ

## 🔧 Основные зависимости

- `numpy` - численные вычисления
- `scipy` - научные вычисления
- `scikit-sparse` - разреженные матрицы
- `PyQt5` - графический интерфейс
- `pyproj` - системы координат и проекции
- `networkx` - работа с графами
- `matplotlib`, `seaborn` - визуализация
- `pandas` - обработка данных

## 🎯 Ключевые особенности

1. **Полная настраиваемость** - любой параметр настраивается
2. **Поддержка форматов приборов** - 15+ форматов (Leica, Sokkia, Trimble, Topcon, и др.)
3. **Математический аппарат** - параметрический метод МНК, робастные методы
4. **Предобработка данных** - 9 этапов с контролем допусков
5. **Анализ надёжности** - теория Баарда, поиск грубых ошибок
6. **Системы координат** - полная база СК РФ
7. **Отчётность** - ведомости и отчёты по ГОСТ 7.32-2017
