# GeoAdjust Pro
GeoAdjust Pro — профессиональная система уравнивания геодезических сетей, предназначенная для обработки реальных полевых геодезических измерений с максимальной точностью и соответствием нормативным требованиям Российской Федерации.
Ключевые особенности
1. Полная настраиваемость
  Любой параметр настраивается: метрология приборов, методы обработки, допуски
  Сохранение всех настроек в проекте для воспроизводимости результатов
  Трехуровневая иерархия настроек: глобальные, проектные, сессионные
2. Поддержка форматов приборов
  15+ форматов: Leica GSI/DAT, Sokkia SDR, Trimble M5/JobXML, Topcon GTS, Nikon, Geodimeter, УОМЗ, и др.
  Автоматическое определение формата файла
  Обработка "сырых" отсчетов без потери метаданных
3. Математический аппарат
  Параметрический метод по Ю.И. Маркузе (классический МНК)
  Робастные методы: Хьюбера, Тьюки, L1-минимизация
  Свободное уравнивание для сетей без исходных пунктов
  Метод Холецкого для решения нормальных уравнений (через scikit-sparse)
4. Предобработка данных
  9 этапов предобработки с контролем 27 инструктивных допусков
  Автоматическое распознавание топологии сети, ходов и секций
  Применение редукций: атмосферные, рефракция, кривизна Земли
5. Анализ надёжности
  Теория надёжности Баарда: внутренняя и внешняя надёжность
  5 методов поиска грубых ошибок: стандартизованные остатки, трассирование ходов, L1-анализ, последовательное отключение ИП, анализ влияния
  Система отключения/восстановления данных
6. Системы координат
  Полная база СК РФ: СК-42, СК-95, ГСК-2011, МСК 128 субъектов
  Преобразования координат: 7 параметров Гельмерта
  Модели геоида: EGM2008, Рус-геоид-2011
  Проекция Гаусса-Крюгера
7. Отчётность
  Ведомости результатов: координаты, поправки в измерения
  Отчёты по ГОСТ 7.32-2017 с автоматической нумерацией формул
  Сертификат соответствия нормативным требованиям
